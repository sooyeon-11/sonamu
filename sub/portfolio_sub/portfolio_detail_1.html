철스크랩 제조 D사

폐자원을 재창조하여 산업의 꽃으로 승화시켜 깨끗한 환경을 조성하는 철스크랩 전문기업

친환경성 설비 개발, 공정설계 및 컨설팅, ICR/BC, 원자재 및 공정 관리, 다양한 금형 설비 제작

프로젝트 개요

서비스 설명
폐자원을 재창조하여 사용할 수 있게 만든 프로그램
사용 기술
C#, Spring, JavaScript, PHP, HTML, CSS, MSSQL
주요 기능
ERP 구축, 무인계근프로그램, SCM, 검수프로그램,
모바일 통합관리 프로그램

요구사항

  • 모바일 연동 모바일 앱을 통해 실시간 단가 조회 가능
  • 실시간 검수 웹 검수프로그램을 통해 무인계근으로 올라온 계근데이터를 실시간으로 검수
  • 무인계근시스템 indicator와 연동하여 실시간 무게정보를 가져오고 cctv 캡쳐를 통해
    차량 및 indicator 무게 이미지와 계근데이터 저장

Our Solution

As-is

  • ✔ 인사, 급여, 회계 등 기존 수기 작성
  • ✔ 기존 비상용 DATABASE 의한 시스템의 불안정
  • ✔ 노후화된 펌프 점검에 따른 위험 요소 발생
  • ✔ 매입/출고 단가 수기 작성

To-be

  • ✔ ERP 구축
  • ✔ 상업용 DATABASE MSSQL DB 도입
  • ✔ 길로틴 펌프 압력센서, 작동유 레벨게이지, 설비 I/F 도입
  • ✔ 모바일 앱 연동 및 태블릿 PC 도입
